# v2.1.4 마이그레이션 가이드

OverrideConfig를 사용하면 API에서 또는 Embed를 사용할 때 플로우 구성을 재정의할 수 있습니다. 보안상의 이유로 현재 기본적으로 비활성화되어 있습니다.

사용자는 UI에서 재정의할 수 있는 구성을 명시적으로 지정해야 합니다.

1.) 구성으로 이동합니다:

<figure><img src="../.gitbook/assets/image (189).png" alt="" width="221"><figcaption></figcaption></figure>

2.) 재정의 구성 활성화:

<figure><img src="../.gitbook/assets/image (190).png" alt=""><figcaption></figcaption></figure>

3.) 재정의할 수 있는 구성의 토글을 켜고 저장합니다.

<figure><img src="../.gitbook/assets/image (191).png" alt=""><figcaption></figcaption></figure>

4.) 예를 들어 사용자는 이러한 변수와 구성을 재정의할 수 있습니다. [OverrideConfig](/broken/pages/F2AfRpI7qYixNiBWpmIe#override-config)를 참조하세요.

```json
{
    "overrideConfig": {
        "systemMessage": "You are helpful assistant",
        "vars": {
            "character": "nice"
        }
    }
}
```
