# Migration Guide

***

Flowise는 일반적으로 역호환성을 제공합니다. 즉, 업데이트는 지속적인 개발 경로를 따릅니다. 그러나 때때로 breaking changes가 있을 수 있습니다.

이 섹션은 다양한 breaking 버전으로 마이그레이션할 때 지침을 제공합니다.
