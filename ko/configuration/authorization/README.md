# 인증

***

이 섹션은 애플리케이션 및 chatflow 수준의 인증 메커니즘에 초점을 맞춘 Flowise를 사용한 보안 구성에 대해 안내합니다.

강력한 인증을 구현함으로써 Flowise 인스턴스를 보호하고 권한 있는 사용자만 chatflow에 액세스하고 상호 작용할 수 있도록 할 수 있습니다.
