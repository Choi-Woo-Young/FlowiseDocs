# 배포

***

Flowise는 플랫폼에 구애받지 않는 아키텍처로 설계되어 인프라 요구 사항에 맞는 다양한 배포 환경과의 호환성을 보장합니다.

## 로컬 머신

Flowise를 로컬로 배포하려면 [Get Started](../../getting-started/) 가이드를 따르세요.

## 현대 클라우드 제공자

현대적인 클라우드 플랫폼은 자동화와 개발자 워크플로우에 초점을 맞추어 클라우드 관리 및 지속적인 유지 관리를 단순화합니다.&#x20;

이는 필요한 기술 전문 지식을 줄이지만 기본 인프라에 대한 사용자 정의 수준을 제한할 수 있습니다.

* [Elestio](https://elest.io/open-source/flowiseai)
* [Hugging Face](hugging-face.md)
* [Northflank](https://northflank.com/stacks/deploy-flowiseai)
* [Railway](railway.md)
* [Render](render.md)
* [Replit](replit.md)
* [RepoCloud](https://repocloud.io/details/?app\_id=29)
* [Sealos](sealos.md)
* [Zeabur](zeabur.md)

## 기존 클라우드 제공자

기존 클라우드 제공자는 특정 요구 사항에 맞게 관리하고 최적화하기 위해 더 높은 수준의 기술 전문 지식이 필요합니다.&#x20;

그러나 이러한 복잡성은 클라우드 환경에 대한 더 큰 유연성과 제어를 제공합니다.

* [AWS](aws.md)
* [Azure](azure.md)
* [DigitalOcean](digital-ocean.md)
* [GCP](gcp.md)
* [Kubernetes using Helm](https://artifacthub.io/packages/helm/cowboysysop/flowise)
