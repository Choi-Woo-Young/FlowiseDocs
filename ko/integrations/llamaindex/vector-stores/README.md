# Vector Stores

***

Vector Store 또는 벡터 데이터베이스는 고차원 수치 벡터를 저장하고 검색하는 데 특화된 데이터베이스 시스템의 한 유형입니다. Vector Store는 이러한 벡터를 효율적으로 관리하고 인덱싱하도록 설계되어 빠른 유사성 검색을 가능하게 합니다.
