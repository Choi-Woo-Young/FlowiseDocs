# LlamaIndex

***

[LlamaIndex](https://www.llamaindex.ai/)는 LLM 애플리케이션이 비공개 또는 도메인별 데이터를 수집, 구조화 및 액세스하기 위한 데이터 프레임워크입니다. RAG(Retrieval Augmented Generation) 앱을 설계하기 위한 고급 검색 기술을 갖추고 있습니다.

Flowise는 시각적 인터페이스를 제공하여 LlamaIndex를 보완합니다. 여기서 노드는 별도의 섹션으로 구성되어 워크플로우를 더 쉽게 구축할 수 있습니다.
