# Agents

***

자체로는 언어 모델은 행동을 취할 수 없습니다. 텍스트만 출력합니다.

Agent는 어떤 행동을 취해야 할지, 그리고 해당 행동의 입력이 무엇이어야 할지 결정하기 위해 LLM을 추론 엔진으로 사용하는 시스템입니다. 이러한 행동의 결과는 agent로 다시 피드백될 수 있으며, agent는 추가 행동이 필요한지 또는 완료해도 되는지 결정합니다.
