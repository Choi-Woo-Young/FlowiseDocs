# Vector Stores

***

Vector store 또는 vector database는 고차원 숫자 벡터를 저장하고 검색하는 데 특화된 데이터베이스 시스템 유형을 말합니다. Vector store는 이러한 벡터를 효율적으로 관리하고 인덱싱하도록 설계되어 빠른 유사도 검색을 가능하게 합니다.
