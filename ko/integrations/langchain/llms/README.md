# LLMs

***

대규모 언어 모델(LLM)은 방대한 텍스트 데이터로 학습된 AI 시스템입니다. 이를 통해 광범위한 프롬프트와 질문에 대응하여 인간과 같은 텍스트로 소통하고 생성할 수 있습니다. 본질적으로 이들은 복잡한 언어를 이해하고 대응할 수 있습니다.
