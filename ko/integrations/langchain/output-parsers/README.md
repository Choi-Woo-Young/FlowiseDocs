# Output Parsers

***

Output Parser node는 모델의 output을 받아 다운스트림 작업에 더 적합한 형식으로 변환하는 역할을 합니다. LLM을 사용하여 구조화된 데이터를 생성하거나, chat model 및 LLM의 output을 정규화할 때 유용합니다.

* [Custom List Output Parser](custom-list-output-parser.md)
