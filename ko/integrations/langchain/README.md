# LangChain

***

[**LangChain**](https://www.langchain.com/)은 언어 모델로 구동되는 애플리케이션을 개발하기 위한 프레임워크입니다. 생성형 AI 애플리케이션을 만들고, 데이터 소스, 벡터, 메모리를 LLM과 연결하는 프로세스를 단순화합니다.

Flowise는 시각적 인터페이스를 제공하여 LangChain을 보완합니다. 여기서 노드는 서로 다른 섹션으로 구성되어 워크플로우를 더 쉽게 구축할 수 있습니다.
