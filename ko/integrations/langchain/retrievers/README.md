# Retrievers

***

Retriever node는 비정형 query가 주어지면 문서를 반환합니다. 이는 vector store보다 더 일반적입니다. Retriever는 문서를 저장할 필요는 없으며, 단지 문서를 반환(또는 검색)하기만 하면 됩니다.
