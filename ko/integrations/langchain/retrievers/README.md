# Retrievers

***

Retriever 노드는 구조화되지 않은 쿼리가 주어졌을 때 문서를 반환합니다. 이는 Vector Store보다 더 일반적입니다. Retriever는 문서를 저장할 필요가 없으며 반환 (또는 검색)하기만 하면 됩니다.
