# Prompts

***

Prompt template node는 사용자 input과 parameter를 언어 모델을 위한 지시사항으로 변환하는 데 도움을 줍니다. 이를 통해 모델의 응답을 안내하고, 모델이 컨텍스트를 이해하여 관련성 있고 일관된 언어 기반 output을 생성하도록 도울 수 있습니다.
