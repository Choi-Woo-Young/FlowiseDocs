# Prompts

***

Prompt 템플릿 노드는 사용자 입력 및 매개변수를 언어 모델에 대한 지시사항으로 변환하는 데 도움이 됩니다. 이를 사용하여 모델의 응답을 안내하고 컨텍스트를 이해하도록 도와 관련성 있고 일관된 언어 기반 출력을 생성하도록 할 수 있습니다.
