# Embeddings

***

임베딩은 부동소수점 숫자 벡터(리스트)입니다. 두 벡터 간의 거리는 관련성을 측정합니다. 작은 거리는 높은 관련성을 의미하고 큰 거리는 낮은 관련성을 의미합니다.

임베딩은 텍스트 데이터의 수치 표현을 만드는 데 사용할 수 있습니다. 이 수치 표현은 유사한 문서를 찾는 데 사용될 수 있기 때문에 유용합니다.

일반적으로 다음 용도로 사용됩니다.

* 검색(결과를 쿼리 문자열과의 관련성으로 순위 지정)
* 클러스터링(텍스트 문자열을 유사성으로 그룹화)
* 추천(관련 텍스트 문자열이 있는 항목 추천)
* 이상 탐지(관련성이 낮은 이상치 식별)
* 다양성 측정(유사성 분포 분석)
* 분류(텍스트 문자열을 가장 유사한 레이블로 분류)
