# Embeddings

***

Embedding은 부동 소수점 숫자의 벡터 (목록)입니다. 두 벡터 간의 거리는 관련성을 측정합니다. 작은 거리는 높은 관련성을 나타내고 큰 거리는 낮은 관련성을 나타냅니다.

Embedding은 텍스트 데이터의 수치 표현을 만드는 데 사용될 수 있습니다. 이 수치 표현은 유사한 문서를 찾는 데 사용할 수 있기 때문에 유용합니다.

일반적으로 다음에 사용됩니다:

* 검색 (결과가 쿼리 문자열과의 관련성으로 순위를 매김)
* 클러스터링 (텍스트 문자열이 유사성으로 그룹화됨)
* 추천 (관련 텍스트 문자열을 가진 항목이 추천됨)
* 이상 탐지 (관련성이 적은 이상치가 식별됨)
* 다양성 측정 (유사성 분포가 분석됨)
* 분류 (텍스트 문자열이 가장 유사한 레이블로 분류됨)
