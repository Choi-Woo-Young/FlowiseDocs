# Langfuse

[Langfuse](https://langfuse.com)는 팀이 API 호출을 추적하고 성능을 모니터링하며 AI 애플리케이션의 문제를 디버깅하는 데 도움이 되는 오픈 소스 LLM 엔지니어링 플랫폼입니다.

네이티브 통합을 통해 Flowise를 사용하여 코드 없이 복잡한 LLM 애플리케이션을 빠르게 만들고 Langfuse를 사용하여 모니터링하고 개선할 수 있습니다.

이 통합은 UI, API 및 임베드를 포함한 Flowise의 모든 사용 사례를 지원합니다.

선택적으로 `release`를 추가하여 흐름의 현재 버전에 태그를 지정할 수 있습니다. 일반적으로 다른 옵션을 변경할 필요는 없습니다.
