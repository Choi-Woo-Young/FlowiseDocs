# 기여 가이드 (Contribution Guide)

***

모든 기여를 환영합니다! 기술 수준이나 기술적 배경에 관계없이 이 프로젝트의 성장을 도울 수 있습니다. 다음은 몇 가지 기여 방법입니다:

## ⭐ Star

[Github Repo](https://github.com/FlowiseAI/Flowise)에 Star를 누르고 공유하세요.

## 🙌 Chatflow 공유

네! Flowise를 어떻게 사용하는지 공유하는 것도 기여의 한 방법입니다. chatflow를 JSON으로 내보내고, 스크린샷을 첨부하여 [Show and Tell 섹션](https://github.com/FlowiseAI/Flowise/discussions/categories/show-and-tell)에 공유하세요.

## 💡 아이디어

새로운 기능, 앱 통합에 대한 아이디어를 환영합니다. [Ideas 섹션](https://github.com/FlowiseAI/Flowise/discussions/categories/ideas)에 제안을 제출하세요.

## 🙋 Q\&A

더 알고 싶으신가요? [Q\&A 섹션](https://github.com/FlowiseAI/Flowise/discussions/categories/q-a)에서 질문에 대한 답변을 검색하세요. 찾을 수 없다면 주저하지 말고 새로운 질문을 만드세요. 비슷한 질문을 가진 다른 사람들에게 도움이 될 수 있습니다.

## 🐞 버그 보고

문제를 발견하셨나요? [보고하세요](https://github.com/FlowiseAI/Flowise/issues/new/choose).

## 📖 Docs에 기여

1. 공식 [Flowise Docs Repo](https://github.com/FlowiseAI/FlowiseDocs)를 Fork하세요
2. Fork한 저장소를 클론하세요
3. 새 Branch를 만드세요
4. 방금 만든 Branch로 전환하세요
5.  저장소 폴더로 이동하세요

    ```bash
    cd FlowiseDocs
    ```
6. 변경 사항을 만드세요
7. 변경 사항을 Commit하고 Fork한 Branch에서 [FlowiseDocs main](https://github.com/FlowiseAI/FlowiseDocs)을 가리키는 Pull Request를 제출하세요

## 👨‍💻 코드에 기여

코드 기여 방법을 알아보려면 [개발자용 (For Developers)](../getting-started/#setup-2) 섹션으로 이동하여 안내를 따르세요.

새로운 node 통합에 기여하는 경우 [Node 빌드하기](building-node.md) 가이드를 읽어보세요.

## 🏷️ Pull Request 프로세스

Pull Request를 열면 FlowiseAI 팀의 구성원이 자동으로 알림을 받거나 할당됩니다. [Discord](https://discord.gg/jbaHfsRVBW)를 통해 연락하실 수도 있습니다.

## 📜 행동 강령 (Code of Conduct)

이 프로젝트와 참여하는 모든 사람은 [파일](https://github.com/FlowiseAI/Flowise/blob/main/CODE\_OF\_CONDUCT.md)에서 확인할 수 있는 행동 강령의 적용을 받습니다. 참여함으로써 이 강령을 준수할 것으로 기대됩니다.

용납할 수 없는 행동은 hello@flowiseai.com으로 신고해 주세요.
