# 기여 가이드

***

모든 기여에 감사합니다! 기술 수준이나 배경에 관계없이 이 프로젝트를 성장시키는 데 도움을 줄 수 있습니다. 기여할 수 있는 몇 가지 방법은 다음과 같습니다:

## ⭐ 별표

[Github Repo](https://github.com/FlowiseAI/Flowise)에 별표를 표시하고 공유하세요.

## 🙌 Chatflow 공유

그렇습니다! Flowise 사용 방법을 공유하는 것도 기여하는 방법입니다. chatflow를 JSON으로 내보내고 스크린샷을 첨부한 후 [Show and Tell section](https://github.com/FlowiseAI/Flowise/discussions/categories/show-and-tell)에서 공유하세요.

## 💡 아이디어

새로운 기능, 앱 통합 아이디어를 환영합니다. 제안을 [Ideas section](https://github.com/FlowiseAI/Flowise/discussions/categories/ideas)에 제출하세요.

## 🙋 Q&A

더 알고 싶으신가요? [Q&A section](https://github.com/FlowiseAI/Flowise/discussions/categories/q-a)에서 질문에 대한 답변을 검색하세요. 찾을 수 없으면 새로운 질문을 작성하세요. 비슷한 질문을 하는 다른 사람들에게 도움이 될 수 있습니다.

## 🐞 버그 보고

문제를 발견했나요? [보고해주세요](https://github.com/FlowiseAI/Flowise/issues/new/choose).

## 📖 문서에 기여

1. 공식 [Flowise Docs Repo](https://github.com/FlowiseAI/FlowiseDocs) 포크
2. 포크된 저장소 복제
3. 새 분기 생성
4. 방금 생성한 분기로 전환
5.  저장소 폴더로 이동

    ```bash
    cd FlowiseDocs
    ```
6. 변경 사항 적용
7. 변경 사항 커밋 및 포크된 분기에서 [FlowiseDocs main](https://github.com/FlowiseAI/FlowiseDocs)을 가리키는 Pull Request 제출

## 👨‍💻 코드에 기여

코드 기여 방법을 배우려면 [For Developers](../getting-started/#setup-2) 섹션으로 이동하여 지침을 따르세요.

새 노드 통합에 기여하는 경우 [Building Node](building-node.md) 가이드를 읽으세요.

## 🏷️ Pull Request 프로세스

Pull Request를 열면 FlowiseAI 팀의 구성원이 자동으로 알림을 받고 할당됩니다. [Discord](https://discord.gg/jbaHfsRVBW)에서 저희에게 연락할 수도 있습니다.

## 📜 행동 강령

이 프로젝트 및 참여하는 모든 사람은 [file](https://github.com/FlowiseAI/Flowise/blob/main/CODE\_OF\_CONDUCT.md)에서 찾을 수 있는 행동 강령으로 관리됩니다. 참여함으로써 이 규범을 유지하기로 동의합니다.

부적절한 행동은 hello@flowiseai.com으로 보고하세요.
